🚀 Primo Utilizzo

1

Incolla uno o più URL

Nel campo di input in alto, incolla gli indirizzi web che vuoi convertire in PDF. Puoi incollare un URL per riga oppure trascinare un file .txt o .csv.

2

Configura le impostazioni (opzionale)

Personalizza la larghezza della pagina, il numero di conversioni parallele, timeout e altri parametri secondo le tue necessità.

3

Scegli la cartella di output

Fai clic su "Sfoglia" per selezionare dove salvare i file PDF generati (di default: Desktop o Documenti).

4

Clicca "Converti"

L'applicazione inizierà la conversione. Vedrai una barra di avanzamento in tempo reale.

5

Recupera i file PDF

Una volta completato, i file PDF saranno nella cartella di output scelta. Clicca "Apri cartella" per accedervi rapidamente.

🔧 Risoluzione dei Problemi

❌ "L'applicazione non parte"

Soluzione: Assicurati di avere Windows 10/11 64-bit. Scarica la versione completa (312 MB) che include Chromium integrato.

❌ "Windows dice che il file è pericoloso"

Soluzione: È un avviso falso (SmartScreen di Windows). Clicca "Altre informazioni" → "Esegui comunque". L'applicazione è sicura.

❌ "La conversione non funziona"

Soluzione: Verifica che tu abbia una connessione internet attiva. Alcuni siti potrebbero bloccare l'accesso automatico — prova con siti pubblici come example.com, wikipedia.org, ecc.

❌ "Il file ZIP è corrotto"

Soluzione: Scarica di nuovo il file. Se il problema persiste, contatta lo sviluppatore sul repository GitHub.

❌ "Spazio disco insufficiente"

Soluzione: Hai bisogno di almeno 750 MB liberi. Libera spazio eliminando file temporanei o vecchi scaricamenti.
